# TinyTot Learning Adventures 🦖🎨

A delightful, engaging, and child-friendly mobile learning application for toddlers (2-6 years old). Inspired by KidloLand, this app focuses on visual immersion, intuitive navigation, and gamified education.

## 🌟 Key Features

- **Child-First UX**: Large touch targets, high-contrast colors, and minimal text for pre-literate users.
- **Interactive Categories**:
  - **ABC Adventures**: Visual identification of fruits and letters.
  - **Number Fun**: Engaging "Find the Number" games.
- **Micro-interactions**: Rewarding animations and visual feedback for every action.
- **Gamification**: A star-based reward system with celebration screens.
- **Safe Interaction**: No accidental clicks or complex navigation paths.

## 🚀 Getting Started

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3 (Modern features like Flexbox/Grid).
- **Logic**: Vanilla JavaScript (ES6+).
- **Design**: AI-generated character and vector assets.

## 🎨 Design Decisions

- **Color Palette**: Uses vibrant, primary colors (Sky Blue, Sun Yellow, Berry Red) to stimulate engagement and provide clear differentiation.
- **Typography**: Uses the 'Fredoka' rounded font for a soft, friendly aesthetic.
- **Navigation**: Replaces text with universally recognizable icons and emojis where possible.
